# Inventory Management System

##  Project Overview

This project is a simple and interactive Inventory Management System developed using HTML, CSS, and JavaScript. It allows users to manage products by adding, editing, deleting, and searching items in the inventory.

The system also calculates the total inventory value automatically and stores data using localStorage so that the products remain saved even after refreshing the browser.



## Features

- Add Product
- Edit Product
- Delete Product
- Search Product
- Automatic Total Calculation
- Total Inventory Value Dashboard
- Product Categories
- Clear All Products
- Data Saving using localStorage
- Responsive and Interactive User Interface



##  Technologies Used

- HTML
- CSS
- JavaScript
